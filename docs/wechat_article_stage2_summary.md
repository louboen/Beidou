## 🐛 遇到的问题及解决方案

### 问题 1：浏览器缓存导致图片内容相同

**表现**：
- 生成了 4 个不同的 HTML 文件
- 但所有截图显示相同内容（清明养生）
- 文件大小完全相同（123701 bytes）

**原因**：
- 浏览器未关闭，缓存了旧页面
- 连续打开多个页面时，浏览器返回缓存内容
- 截图工具获取的是缓存的页面

**解决**：
```bash
# 1. 关闭浏览器
openclaw browser stop

# 2. 清除缓存
rm -f /home/admin/.openclaw/media/browser/*.png

# 3. 重启浏览器
openclaw browser start

# 4. 逐一生成（不要批量）
browser.open() → sleep(3) → browser.screenshot()
```

**记录位置**：`wechat_cover_generation_guide.md` - 常见问题 Q6

---

### 问题 2：emoji 不显示

**表现**：
- HTML 中使用了 🌿 🍃 🌱 等 emoji
- 截图中显示为方框或空白

**原因**：
- 浏览器字体不支持 emoji
- 截图时 emoji 渲染失败

**解决**：
- 使用 CSS 图形代替 emoji
- 使用 SVG 图形
- 使用 CSS 径向渐变

---

### 问题 3：HTTP 服务器连接失败

**表现**：
- 启动 HTTP 服务器后无法访问
- 浏览器显示"ERR_CONNECTION_REFUSED"

**原因**：
- 服务器进程被终止
- 端口被占用
- 防火墙阻止

**解决**：
```bash
# 1. 检查服务器状态
ps aux | grep 'http.server'

# 2. 关闭旧服务器
pkill -f 'http.server'

# 3. 启动新服务器
python3 -m http.server 9000 &

# 4. 验证服务器
curl -s http://localhost:9000/file.html | head -5
```
