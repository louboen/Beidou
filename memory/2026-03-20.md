## 微信公众号文章发布系统 - 封面生成问题修复

### 问题描述
- **现象**：生成的封面图片与文章内容不匹配
- **原因**：使用 `subprocess` 调用 `openclaw browser screenshot` 命令时，浏览器页面未正确加载
- **表现**：截图尺寸为 780x441（应为 900x500），内容显示不完整

### 解决方案
1. **正确流程**：
   - 启动 HTTP 服务器提供 HTML 文件
   - 使用 `browser.open()` 打开页面（等待加载完成）
   - 使用 `browser.screenshot()` 截图
   - 获得完整尺寸的封面图片（900x500）

2. **关键修复**：
   - 直接使用 `browser` 工具而非 subprocess
   - 确保页面完全加载后再截图
   - 验证生成的图片尺寸和文件大小
