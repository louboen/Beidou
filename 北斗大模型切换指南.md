# 北斗大模型切换指南

> 适用于 Win11 OpenClaw 环境，通过 Tailscale/Headscale 中继访问

---

## 📋 快速切换

### 切换到通义千问 Qwen3.5
```powershell
# 1. 更新 API 密钥（dashscope 阿里云）
$c = Get-Content ~/.openclaw/openclaw.json | ConvertFrom-Json
$c.models.providers.dashscope.apiKey = "sk-YOUR_KEY_HERE"
$c.agents.defaults.model.primary = "dashscope/qwen3.5-plus"
$c | ConvertTo-Json -Depth 10 | Set-Content ~/.openclaw/openclaw.json -NoNewline

# 2. 重启网关
openclaw gateway restart
```

### 切换到智谱 GLM-4
```powershell
# 1. 更新配置
$c = Get-Content ~/.openclaw/openclaw.json | ConvertFrom-Json
$c.agents.defaults.model.primary = "zai/glm-4-flash"
$c | ConvertTo-Json -Depth 10 | Set-Content ~/.openclaw/openclaw.json -NoNewline

# 2. 重启网关
openclaw gateway restart
```

### 切换到本地 Ollama 模型
```powershell
# 切换到 Qwen2.5 7B
$c = Get-Content ~/.openclaw/openclaw.json | ConvertFrom-Json
$c.agents.defaults.model.primary = "ollama/qwen2.5:7b"
$c | ConvertTo-Json -Depth 10 | Set-Content ~/.openclaw/openclaw.json -NoNewline

openclaw gateway restart
```


```json
{
  "version": 1,
  "profiles": {

  }
```

### 安装新模型
```powershell
# 拉取模型
ollama pull qwen2.5:7b
ollama pull llama3.2:3b
ollama pull deepseek-r1:8b

# 查看已安装
ollama list

# 删除模型
ollama rm <model-name>
```

### 检查 Ollama 服务
```powershell
# 服务状态
Get-Service Ollama

# 启动服务
Start-Service Ollama

# 测试连接
curl http://localhost:11434/api/tags
```

## 🔧 完整配置流程

### 1. 查看当前模型
```powershell
openclaw models list
```

### 2. 查看可用模型
```powershell
openclaw models list --all
```

### 3. 修改配置
```powershell
# 方法一：直接编辑 JSON
notepad C:\Users\loubo\.openclaw\openclaw.json

# 方法二：PowerShell 命令
$c = Get-Content ~/.openclaw/openclaw.json | ConvertFrom-Json
$c.agents.defaults.model.primary = "模型 ID"
$c | ConvertTo-Json -Depth 10 | Set-Content ~/.openclaw/openclaw.json -NoNewline
```

### 4. 配置 API 密钥
```powershell
notepad C:\Users\loubo\.openclaw\agents\main\agent\auth-profiles.json
```

### 5. 重启网关
```powershell
openclaw gateway restart
```

### 6. 验证配置
```powershell
openclaw status
openclaw models list
```

## ⚠️ 故障排查

### 网关无法启动
```powershell
# 查看日志
openclaw logs --follow

# 检查配置
openclaw doctor --fix

# 强制重启
openclaw gateway stop
Start-Sleep -Seconds 3
openclaw gateway start
```

### API 密钥错误
```powershell
# 验证 auth-profiles.json
cat C:\Users\loubo\.openclaw\agents\main\agent\auth-profiles.json

# 确保 lastGood 指向正确的 provider
```

### 模型列表为空
```powershell
# 刷新模型缓存
openclaw models refresh

# 检查 Provider 配置
openclaw models providers list
```

## 🌐 远程访问

### 通过 Tailscale IP 访问 Win11
```bash
# 从 Linux 服务器 SSH
ssh -i ~/.ssh/id_ed25519 loubo@100.64.0.2

# 访问 Control UI（需修改网关绑定为 0.0.0.0）
http://100.64.0.2:18789/
```

### 修改网关绑定地址（可选）
```powershell
$c = Get-Content ~/.openclaw/openclaw.json | ConvertFrom-Json
$c.gateway.bind = "0.0.0.0"
$c | ConvertTo-Json -Depth 10 | Set-Content ~/.openclaw/openclaw.json -NoNewline
openclaw gateway restart
```
