### 方案二：检查微信公众号权限

#### 需要检查的项目
1. **接口权限**
   - 登录微信公众平台
   - 进入「设置」→「公众号设置」→「功能设置」
   - 检查「网页授权域名」和「JS接口安全域名」

2. **IP白名单**
   - 进入「设置」→「安全中心」
   - 检查服务器 IP 是否在白名单中
   - 当前服务器 IP: 需要获取

3. **API权限**
   - 确保有「草稿箱管理」权限
   - 检查「基本配置」中的「IP白名单」

#### 操作步骤
```bash
# 获取服务器公网 IP
curl ifconfig.me
# 或
curl ipinfo.io/ip
```

将获取的 IP 添加到微信公众平台的 IP 白名单中。

### 方案三：使用微信测试号

#### 优点
- 不受正式号权限限制
- 可以测试所有 API 功能
- 快速验证问题

#### 步骤
1. 访问: https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login
2. 使用微信扫码登录
3. 获取测试号的 AppID 和 AppSecret
4. 更新 `.env` 文件中的配置
5. 重新测试发布功能
