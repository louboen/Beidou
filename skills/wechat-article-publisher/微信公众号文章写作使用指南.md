# 微信公众号文章写作使用指南

## 📋 快速开始

### 1. 创建文章
```bash
cd ~/.openclaw/workspace/skills/wechat-article-publisher

# 查看可用主题
python3 scripts/write_wechat_article.py --list-topics

# 创建中医养生文章
python3 scripts/write_wechat_article.py --topic 中医养生

# 创建中医食疗文章
python3 scripts/write_wechat_article.py --topic 中医食疗

# 创建穴位保健文章
python3 scripts/write_wechat_article.py --topic 穴位保健
```

### 2. 查看生成的文件
每次运行会生成两个文件：
- **Markdown 文件** (`*.md`): 用于编辑和存档
- **HTML 文件** (`*.html`): 用于微信公众号发布

### 3. 手动发布到微信公众号

#### 步骤：
1. **登录微信公众平台**
   - 访问: https://mp.weixin.qq.com
   - 使用账号密码登录

2. **创建新文章**
   - 点击"内容与互动" → "草稿箱"
   - 点击"新建图文"

3. **粘贴内容**
   - 打开生成的 HTML 文件
   - 全选复制内容
   - 粘贴到微信公众号编辑器

4. **添加封面图片**
   - 点击"封面"区域
   - 上传合适的图片（建议 900x500 像素）
   - 选择"从正文选择"或"本地上传"

5. **完善信息**
   - 填写标题（可修改）
   - 填写作者（可修改）
   - 填写摘要（可修改）
   - 选择分类标签

6. **预览和发布**
   - 点击"预览"检查效果
   - 点击"保存为草稿"或"群发"

## 📊 微信公众号要求

### 内容限制
- **标题**: ≤64 字符
- **作者**: ≤8 字符
- **摘要**: ≤120 字符
- **内容**: 支持 HTML，避免复杂样式

### 图片要求
- **封面图片**: 900x500 像素，JPG/PNG 格式
- **内容图片**: ≤2MB，建议宽度 900px
- **图片数量**: 单篇文章最多 20 张

### 发布建议
1. **发布时间**: 上午 8-10 点，下午 6-8 点
2. **发布频率**: 每周 2-3 篇为宜
3. **内容规划**: 系列化、专题化

## 🔍 常见问题

### Q1: 文章格式有问题怎么办？
A: 使用生成的 HTML 文件，避免手动调整格式。

### Q2: 封面图片上传失败？
A: 确保图片尺寸为 900x500 像素，JPG/PNG 格式。

### Q3: 如何添加更多图片？
A: 在微信公众号编辑器中点击"图片"按钮上传。

### Q4: 文章可以修改吗？
A: 可以，在微信公众号编辑器中直接修改。

### Q5: 如何查看发布效果？
A: 在微信公众平台查看阅读量、点赞数等数据。

## 📈 效果评估

### 关键指标
1. **阅读量**: 文章被阅读的次数
2. **点赞数**: 读者点赞的数量
3. **分享数**: 文章被分享的次数

3. **发布时间**: 选择读者活跃时段
4. **互动引导**: 鼓励留言和分享
