## 🔧 上传步骤

### 方法一：微信后台手动上传（推荐）

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入 **素材管理** → **图片**
3. 点击 **上传**，选择 `doctor_profile.jpg` 和二维码图片
4. 上传成功后，点击图片获取 **图片链接**
5. 将链接复制到文章模板中

### 方法二：使用 API 自动上传

```bash
cd /home/admin/.openclaw/workspace/skills/wechat-article-publisher

# 上传医生照片
python3 -c "
import requests
import json

# 获取 access_token
appid = 'wx1a0fadc458656bef'
appsecret = '8640812d15d97219575da73caef1e80e'
token_url = f'https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid={appid}&secret={appsecret}'
token = requests.get(token_url).json()['access_token']

# 上传永久素材
upload_url = f'https://api.weixin.qq.com/cgi-bin/material/add_material?access_token={token}&type=image'
with open('doctor_profile.jpg', 'rb') as f:
    files = {'media': f}
    result = requests.post(upload_url, files=files).json()
    print('医生照片上传成功:', result)
"

# 二维码同样方式上传
```
