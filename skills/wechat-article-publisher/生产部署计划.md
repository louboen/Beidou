## 🔧 故障排除指南

### 常见问题及解决方案

#### 1. 发布失败

**症状**: 文章发布失败，返回错误代码

**排查步骤**:
```bash
# 1. 检查 access_token
python3 check_token.py

# 2. 检查文章编码
python3 check_encoding.py articles/article.html

# 3. 检查封面图片
python3 check_cover.py covers/cover.jpg

# 4. 查看详细日志
tail -f publish_logs/$(date +%Y-%m-%d).log
```

**常见错误**:
- `40001`: access_token 无效 → 重新获取
- `40007`: media_id 无效 → 使用永久素材
- `45003`: 标题超长 → 自动截断
- `45110`: 作者超长 → 自动截断

#### 2. 文章乱码

**症状**: 微信显示 `\u4e2d\u533b` 等乱码

**解决方案**:
```bash
# 检查脚本是否使用 ensure_ascii=False
grep -n "ensure_ascii" auto_publish_final_fixed.py

# 应该显示：
# json_data = json.dumps(data, ensure_ascii=False)
```

#### 3. Cron 任务未执行

**症状**: 定时任务未自动运行

**排查步骤**:
```bash
# 1. 检查 Cron 服务状态
```
