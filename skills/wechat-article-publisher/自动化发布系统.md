## 🔍 故障排除

### 常见问题
1. **错误 40007: invalid media_id**
   - **原因**: 使用了临时素材的 `media_id`
   - **解决**: 使用永久素材接口上传图片

2. **错误 45110: author size out of limit**
   - **原因**: 作者名超过 8 字符
   - **解决**: 系统自动截断为 8 字符

3. **错误 45009: API 调用频率限制**
   - **原因**: 调用太频繁
   - **解决**: 添加延迟 `time.sleep(3)`

4. **错误 40113: unsupported file type**
   - **原因**: 图片格式不支持
   - **解决**: 使用 JPG 或 PNG 格式
