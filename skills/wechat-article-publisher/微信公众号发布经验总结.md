### 问题 2: 封面图片上传失败

**错误代码**: 53402 - 封面裁剪失败

**原因**: 
- 图片尺寸不符合要求
- 图片格式不支持

**微信官方要求**:
- 尺寸：900x500 像素
- 格式：JPG/PNG
- 大小：≤2MB

**解决方案**:
- ✅ 支持网络图片 URL 作为封面
- ✅ 自动上传图片获取 media_id
- ✅ 使用 `https://picsum.photos/900/500` 等图床服务

### 问题 3: 标题长度超限

**错误代码**: 45003 - title size out of limit

**原因**: 微信按**字节**计算长度，中文占 3 字节，标题限制 64 字节（约 21 个汉字）。

**解决方案**:
```python
# 按字节截断标题
while len(title.encode('utf-8')) > 64 and len(title) > 0:
    title = title[:-1]
```

**其他字段限制**:
| 字段 | 限制 |
|------|------|
| 标题 | ≤64 字节 |
| 作者 | ≤8 字节 |
| 摘要 | ≤128 字节 |
| 内容 | ≤20000 字节 |

### 问题 4: HTML 内容处理不当

**现象**: 包含完整 `<html>`, `<head>`, `<body>` 标签的文章发布后格式异常。

**原因**: 微信 API 对嵌套 HTML 标签处理不当。

**解决方案**:
```python
# 提取 body 内容，去除外层标签
body_match = re.search(r'<body[^>]*>(.*?)</body>', content, re.DOTALL | re.IGNORECASE)
if body_match:
    html_content = body_match.group(1)
```

**最佳实践**:
- ✅ 只保留微信支持的 HTML 标签（`h1/h2/h3/p/strong/hr`）
- ✅ 移除 `<style>` 标签和复杂 CSS
- ✅ 使用简单内联样式

### 问题 5: 中文编码乱码

**现象**: 发布后内容显示为 `\u6625\u5b63\u4e2d\u533b...` (Unicode 转义序列)。

**原因**: JSON 序列化时使用了默认 ASCII 编码，中文被转义。

**解决方案**:
```python
# 1. 文件读取使用 utf-8-sig 编码
with open(content_path, "r", encoding="utf-8-sig") as f:
    content = f.read()

# 2. JSON 序列化设置 ensure_ascii=False
json_data = json.dumps(data, ensure_ascii=False).encode('utf-8')

# 3. 明确指定 Content-Type
headers = {'Content-Type': 'application/json; charset=utf-8'}
```

## 📊 微信 API 接口

### 1. 获取 access_token
```
GET https://api.weixin.qq.com/cgi-bin/token
参数：grant_type=client_credential&appid=xxx&secret=xxx
```

### 2. 上传封面图片
```
POST https://api.weixin.qq.com/cgi-bin/material/add_material
参数：access_token, type=image
```

### 3. 创建草稿
```
POST https://api.weixin.qq.com/cgi-bin/draft/add
参数：access_token
Body: {"articles": [{...}]}
```

### 4. 发布草稿（可选）
```
POST https://api.weixin.qq.com/cgi-bin/freepublish/submit
参数：access_token, media_id
```
