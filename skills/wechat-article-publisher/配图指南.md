# 中医养生文章配图指南

## 📋 配图原则

1. **相关性**：图片内容必须与文章主题相关
2. **专业性**：图片风格专业、清晰
3. **美观性**：色彩协调，构图美观
4. **版权**：使用可商用图片

## 🛠️ 图片处理流程

### 步骤 1: 生成/选择图片
根据文章内容选择或生成图片

### 步骤 2: 调整尺寸
```bash
# 使用 ImageMagick 调整尺寸
convert input.jpg -resize 900x500^ -gravity center -extent 900x500 output.jpg
```

### 步骤 3: 添加标题（可选）
```bash
# 使用 ImageMagick 添加文字
convert output.jpg \
  -font DejaVu-Sans-Bold -pointsize 48 \
  -fill white -gravity center \
  -annotate 0 "文章标题" \
  final_cover.jpg
```

### 步骤 4: 上传微信
```bash
# 使用发布脚本上传
python3 scripts/publish_official.py \
  --html 文章.html \
  --cover final_cover.jpg
```
