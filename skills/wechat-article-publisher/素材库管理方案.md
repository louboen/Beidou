## 🛠️ 素材管理工具

### 批量重命名脚本
```bash
#!/bin/bash
# 批量重命名图片为标准格式
for img in *.jpg; do
    convert "$img" -resize 900x500^ -gravity center -extent 900x500 "cover_${img}"
done
```

### 图片上传脚本
```bash
#!/bin/bash
# 批量上传到微信素材库
python3 scripts/batch_upload_images.py --folder 封面图库/节气养生/
```

## 🎯 封面选择原则

### ✅ 推荐
- 图片与文章标题强相关
- 色彩与内容主题匹配
- 风格统一，形成品牌感
- 清晰度高，无水印

### ❌ 避免
- 图片与内容无关
- 过于花哨或复杂
- 文字过多过密
- 风格杂乱
