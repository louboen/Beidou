# OpenClaw 微信公众号发布测试

## 测试文章

这是通过 **OpenClaw AI 助手** 自动发布的微信公众号测试文章。

### 功能特点

1. **自动化发布** - 通过 API 直接发布到微信公众号草稿箱
2. **格式支持** - 支持 Markdown 和 HTML 格式
3. **图片上传** - 自动上传图片到微信公众号
4. **快速发布** - 无需手动登录微信公众平台

### 技术架构

- **Python API 客户端** - 处理微信公众号 API 调用
- **环境变量配置** - 安全存储 API 密钥
- **格式转换** - 自动将 Markdown 转换为微信公众号格式

### 使用步骤

1. 配置 `WECHAT_API_KEY` 环境变量
2. 获取微信公众号 AppID
3. 运行发布命令
4. 在微信公众平台预览并发布

### 注意事项

- 文章仅保存到草稿箱，需要手动发布
- 图片需要可公开访问的 URL
- 标题限制在 64 字符以内
