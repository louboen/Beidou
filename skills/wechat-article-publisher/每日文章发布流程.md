## ✅ 选择发布流程

### 第一步：查看当日文章
```bash
# 查看生成的文章列表
cat 每日文章记录/2026-03-19.json
```

**输出示例**:
```json
{
  "date": "2026-03-19",
  "articles": [
    {
      "type": "中医食疗",
      "title": "失眠怎么办？试试这 3 款安神汤",
      "filename": "2026-03-19_中医食疗_失眠怎么办？试试这 .html"
    },
    {
      "type": "穴位保健",
```

### 第二步：预览文章内容
```bash
# 在微信后台预览文章
1. 登录微信公众平台
2. 进入草稿箱
3. 查看已生成的文章
```

### 第三步：选择并发布
```bash
# 选择 1 篇最佳文章发布
python3 scripts/publish_official.py \
  --html articles/2026-03-19_中医食疗_失眠怎么办？试试这 .html \
  --author "娄医生" \
  --cover "封面图片 URL"
```

## 🎨 封面图片生成

### 批量生成封面
```bash
# 为当日文章生成封面提示词
python3 scripts/generate_cover.py \
  --type "中医食疗" \
  --subtype "养生茶" \
  --title "失眠安神汤" \
  --save-prompt
```

### 使用示例
1. 复制提示词到 Midjourney
2. 生成 900x500 图片
3. 保存到素材库对应目录
4. 上传到微信素材库

## 🎯 选择标准

### 优先发布
- ✅ 与当前季节/热点相关
- ✅ 内容实用性强
- ✅ 标题吸引力高
- ✅ 封面图片精美

### 备选发布
- ⏸️ 留作周末或节假日发布
- ⏸️ 内容相似时选择最佳
- ⏸️ 特殊日期调整主题
